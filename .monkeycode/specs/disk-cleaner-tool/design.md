- дҪҝз”Ё `ConcurrentBag<T>` йҒҝе…Қй”Ғз«һдәү

### 9.2 UI жҖ§иғҪдјҳеҢ–

- ж–Үд»¶еҲ—иЎЁдҪҝз”ЁиҷҡжӢҹеҢ–пјҲ`ListView` й»ҳи®Өж”ҜжҢҒпјү
- еӣҫзүҮ/еӣҫж ҮејӮжӯҘеҠ иҪҪ
- иҝӣеәҰжӣҙж–°иҠӮжөҒпјҲ100ms й—ҙйҡ”пјү
- дҪҝз”Ё `DispatcherQueue.TryEnqueue` йҒҝе…Қи·ЁзәҝзЁӢи®ҝй—®

### 9.3 еҶ…еӯҳдјҳеҢ–

- жү«жҸҸз»“жһңеҲҶйЎөеҠ иҪҪпјҲжҜҸйЎө 1000 йЎ№пјү
- еӨ§ж–Үд»¶еҲ—иЎЁдҪҝз”ЁеўһйҮҸеҠ иҪҪ
- еҸҠж—¶йҮҠж”ҫжңӘдҪҝз”Ёзҡ„иө„жәҗпјҲ`IDisposable` жЁЎејҸпјү
